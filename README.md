# DSA-Visualizer-Problem-Solver-in-Python

A step-by-step visualizer for core Data Structures 
and Algorithms, built from scratch in Python.

Algorithms covered:
- Bubble Sort, Selection Sort, Merge Sort
- Binary Search, Linear Search

Data Structures covered:
- Stack, Queue, Singly Linked List

Each algorithm prints its state at every step
so you can see exactly how it works.

Time and Space complexity documented for every algorithm.

Language: Python 3
